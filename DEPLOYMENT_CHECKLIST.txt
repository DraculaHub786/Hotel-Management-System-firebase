================================================================================
COMMON ISSUES & FIXES
================================================================================

Issue: "Firebase credentials file not found"
Fix: Set FIREBASE_CREDENTIALS_PATH in .env or copy file to working directory

Issue: "Secret key changes on restart (development only)"
Fix: Normal in development, set SECRET_KEY in production

Issue: "Chatbot API timeout"
Fix: Normal with free tier, uses keyword fallback automatically

Issue: "bcrypt not found"
Fix: Run pip install -r requirements.txt

Issue: "Password verification fails"
Fix: Old SHA-256 passwords not compatible with bcrypt, users must reset
